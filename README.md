# Java FSE - Week 2

## Overview
This repository contains the completed hands-on exercises and projects for Week 2 of the Java Full Stack Engineer (FSE) Deep Skilling Program.

## Modules Covered

### Module 5: Spring Core and Maven
Completed hands-on exercises:

1. Basic Spring Application
2. Dependency Injection
3. Spring AOP Logging
4. Maven Project Configuration
5. Spring IoC Container
6. Annotation-Based Configuration
7. Constructor and Setter Injection
8. Basic AOP
9. Spring Boot Application

### Module 6: Spring Data JPA with Spring Boot and Hibernate
Completed projects and exercises:

1. orm-learn
   - Spring Data JPA Quick Example
   - Country Management Services
   - Query Methods
   - CRUD Operations

2. orm-learn-employee
   - Employee, Department and Skill Mapping
   - One-to-Many Relationship
   - Many-to-One Relationship
   - Many-to-Many Relationship
   - Hibernate ORM Concepts

3. EmployeeManagementSystem
   - Spring Data JPA
   - CRUD Operations
   - Query Methods
   - Pagination and Sorting
   - Entity Auditing
   - Projections
   - Data Source Configuration
   - Hibernate Features
